Des questions ?

Les playlists sont un excellent moyen de sauvegarder votre collection musicale, pour vous ou pour les partager.

Pour en créer une :

  • 1. Appuyez sur la Bibliothèque.
  • 2. Appuyez sur CRÉER.
  • 3. Donnez un nom à votre playlist.
  • 4. Commencez à ajouter des titres (et nous vous en proposerons des similaires).
  • 1. Appuyez sur Accueil.
  • 2. Appuyez sur Préférences.
  • 3. Appuyez sur Économiseur de données.
  • 4. Activez l'Économiseur de données.

Toutes les playlists avec l'icône Lecture aléatoire seront diffusées en mode de lecture aléatoire.

Certaines playlists ne comportent pas l'icône Lecture aléatoire, vous pouvez donc écouter les titres de votre choix.

Appuyez sur Rechercher. Sous Parcourir tout, appuyez sur Podcasts.
